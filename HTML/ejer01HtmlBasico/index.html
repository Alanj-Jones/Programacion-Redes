Cuerpo principal del contenido

La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado del contenido dentro de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento.
El contenido escrito aqui dentro no se repite en el resto del sitio.
Conviene usar aqui textos que ayudan a encontrar la pagina luego de procesos de indexacion de los buscadores.

Seccion de etiquetas semantica estructurales mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Reglas para etiquetas semanticas:

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. O sea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en todo el documento y corresponde a los elementos de navegacion del documento
  • Reservar la etiqueta H1 para usar dentro de etiquetas top level como main, header o footer.
